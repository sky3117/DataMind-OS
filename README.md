# DataMind OS

An AI-powered autonomous data intelligence platform for intelligent data analysis and insights.

## Features
- File Upload with drag & drop UI
- Data Profiling with comprehensive analysis
- AI Chat on Data using Claude API
- Auto Dashboard with visualizations

## Tech Stack
- Frontend: Next.js 14 + TypeScript + Tailwind CSS
- Backend: Python FastAPI
- Database: PostgreSQL + Redis
- AI: Anthropic Claude API
- Data Engine: DuckDB + Pandas
